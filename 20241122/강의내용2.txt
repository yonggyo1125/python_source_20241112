1. HTML 구조
- 태그로 구성된 문서 
- <태그명>텍스트 또는 다른 태그</태그명>
- HTML 문서 구조
<!DOCTYPE html>
<html>
	<head></head> : 보이지 않는 부분 정의 - 문서의 정보, 문서의 자원 
	<body></body> : 보이는 부분 
</html>

1) 태그 이름
2) 속성명
	<태그 이름=값  이름=값>...</태그>
		이름 - 속성  : 태그에 있는 추가 정보
		속성은 주로 HTML 태그의 기능과 관련된 경우가 많다!
		- href="주소" : a 태그를 클릭 했을 때 이동할 주소에 대한 정보 
		- src="경로" : img 태그의 이미지 경로..
		
	- id, class : 특정 태그를 선택하기 위한 정보로 사용되는 속성!(선택만을 위한 속성!)
    - id : 하나의 요소만을 특정하기 위한 구분 정보, 하나만 선택 가능
    - class : 여러 요소를 특정하기 위한 구분 정보, 여러개 선택 가능 
	- 선택의 목적은 스타일링(CSS)
    - 속성값과 내용
