	작업 모드 - r(읽기), w("쓰기"), a("추가"), b(바이너리)
	write(데이터)
	read() :  파일 내용 전체를 읽어온다
	close() - 자원해제

	readline() : 한줄 한줄 읽어올때 사용
					다 읽게 되면 빈 문자열 ''
					
	readlines() : 모든 행을 리스트 객체로 반환 
	
	open() 반환값 객체로 직접 행별로 읽어 올수 있다!
	
	with : 자동으로 자원해제
	
외부 라이브러리
1. 패키지 저장소 - pypi.org
2. 설치 
pip install 패키지명  : 패키지 설치 
	- pip install 패키지명 == 버전  : 특정 버전의 패키지를 설치 
	- pip install --upgrade 패키지명 : 현재 설치된 패키지를 최신 버전으로 업그레이드 
	
pip uninstall 패키지명 : 패키지 제거 

pip list  : 현재 설치된 패키지의 목록 확인 


버전 체계
32.       5.          21
Major   Minor    Patch

Major 버전 : 하위 버전과 호환성 유무,  32 - 31, 33, 30 ... 다른 버전과는 호환이 안될 가능성이 매우 높다
Minor 버전 : 현재 Major 버전의 다른 버전과 호환성에는 문제가 없는 기능 추가
Patch 버전 : 오류, 수정 보완이 될때 마다 버전이 올라간다
