21. open
	- 파일을 읽고, 쓸때 사용, 
	- 파일 작업을 하려면 파일을 불러와야 한다! / 파일을 불러오면 일정 부분을 메모리에 기록(자원 소비가 필요)
	- 작업이 끝나도 자동으로 메모리를 해제 하지 않는다!
	- 작업이 끝나면 메모리를 직접 해제 함수를 호출 close() / 자원 해제
	- read() : 읽기 - 파일 전체 읽기
	- write(데이터) : 쓰기  
	
	
	- open("파일 경로", "처리 모드")
	
		처리 모드 : 
			r : 파일 읽기
			w : 파일 쓰기, 파일 경로에 파일이 없으면 새로 생성!
			a : 기존 기록된 데이터를 삭제하지 않고 끝에 추가
			b : 바이너리, 텍스트 이외의 데이터인 경우
